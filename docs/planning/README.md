# Planning Notes

Session-handoff notes for picking up where you left off after a break. Not a backlog, not a tracker — just enough context to re-orient quickly.

## Conventions

- **File names start with `YYYY-MM-DD`** (e.g., `2026-05-07-orientation-and-next-steps.md`).
- **Disposable by design.** Once a note has been acted on, feel free to move, fold into another artifact, or delete it.
- **No format requirements.** Write whatever helps future-you get back up to speed — open questions, candidate next steps, what was in progress, what to avoid.

## What belongs here

- Post-break orientation notes
- "Where I left off" session handoffs
- Short-lived plans that don't warrant an ADR or prompt

## What does not belong here

- Architectural decisions (use `docs/decisions/`)
- Implementation prompts (use `docs/prompts/`)
- Retrospectives (use `docs/retrospectives/`)
